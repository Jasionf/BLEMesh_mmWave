官方解释：ESP系列芯片最多支持三种射频收发模块: Bluetooth（BT 和 BLE）, IEEE802.15.4 和 Wi-Fi, 而每款芯片只支持一路被多个射频收发模块共享的 RF，不同模块无法同时使用 RF 收发数据，因此采用时分复用的方法调节不同模块的数据包收发。

技术关键：采用时分复用(简单来说快速切换)
    考虑的因素：
        - 数据是否可以完整发送和接收
        - 切换的触发条件是什么
        - 切换的频率
